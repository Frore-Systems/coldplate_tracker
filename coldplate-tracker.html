COLDPLATE TRACKER

Connect to SharePoint — one-time setup

1
Register a free Azure app

Go to portal.azure.com → App registrations → New.
Name it anything. Set Supported account types to Accounts in this organizational directory only.
Under Redirect URI choose Single-page application and paste the full URL where this file is hosted.
Copy the Application (client) ID.

2
Find your SharePoint site URL & folder

Open the SharePoint folder where data should live. Copy your site URL (e.g. https://company.sharepoint.com/sites/mysite) and the folder path inside the document library (e.g. /Shared Documents/Coldplate Lab).

3
Paste your details below and sign in
Settings are saved in your browser.
Data is read/written directly to your SharePoint folder as coldplate-data.json.
